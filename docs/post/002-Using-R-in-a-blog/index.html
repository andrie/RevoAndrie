You can do many things with R, including writing blogs and blog posts.

To build this site, I am using the Hugo static website generator, and the blogdown package.

Using R and embedding code in the text

This means I can embed R code directly in the text.

The blogdown::serve_site() function then renders the content automatically (by first running rmarkdown and then letting Hugo buidl the site).

So I can do this:

1:100

Including Plots

You can also embed plots:

par(mar = c(0, 1, 0, 1))
pie(
  c(280, 60, 20),
  c('Sky', 'Sunny side of pyramid', 'Shady side of pyramid'),
  col = c('#0292D8', '#F7EA39', '#C4B632'),
  init.angle = -50, border = NA
)
