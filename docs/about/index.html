About Andrie

Andrie de Vries is a data scientist, author and R programmer.

Andrie is co-author of the book R for dummies, the book that helps everyone getting started with R, regardless of prior programming knowledge. He is also a regular contributor to the question and answer site StackOverflow.

He is also a regular speaker at R and industry events, including:

  • UseR!2016: Tutorial - Using R with Jupyter notebooks for reproducible research
  • Machine Learning and Data Science conference, Bangalore 2016: Microsoft R Server & Applications: Comprehensive Overview
  • EARL London, 2015: Reproducible Data Analysis with Revolution R Open (agenda)
  • UseR!2015: The network structure of CRAN
  • Apache Big data conference, Budapest 2015: R as a language for big data analytics.
  • JSM 2015: The network structure of CRAN.
  • EARL London, 2014: Taking a mini CRAN into your organization (agenda).
  • UseR!2013: Using survival analysis for marketing attribution).
